내가 중간 결재자
 - 조건 : 내가 중간 결재자인거 다 가져옴

내가 최종 결재자
- 조건 : 내가 최종결재자 + 중간결재 현황이 y 인것만

WHERE
   (중간결재자 = 나) // 내가 중간결재자인 경우
   OR
   (최종결재자 = 나 AND 중간결재현황 = 진행) // 중간결재자 있고 내가 최종 결재자
   OR
   (최종결재자 = 나 AND 중간결재자 = NULL) // 중간결재자 없이 내가 최종 결재자인 경우

 
IF 중간 NULL 이고 최종 = 대기  > 최종 표시
IF 중간 NOT NULL  이고 최종이 대기면 > 중간 표시
최종이 승인 OR 반려 > 최종 표시'

중간결재자 처리를 디폴트로 0(중간결재자 없음)을 준다
중간결재자를 추가하면 중간결재자 처리를 1(중간결재자 존재)로 바꾼다 
중간결재자는 자신의 사번과 일치하는 모든 결재가져옴  -- 내가 중간 결재자인 경우

중간결재자가 결재를 승인하면 중간결재자 처리가 2(중간결재자 승인)로 바꿈
		  반려하면 3(중간결재자 반려)으로 바꿈


(최종결재자는 자신의 사번과 일치)AND
(0 인 항목을 조회 -- 중간결재자 없이 내가 최종결재자
OR
2 인 항목 조회 -- 중간결재자가 승인 내린 결재)

0 일때 기안		0일때 승인 value = 3 
1 일때 기안 		1일때 승인 value = 2
2 일때 진행
3 일때 반려
